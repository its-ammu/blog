This post exercises the common Markdown elements so you can see how the theme
renders a real article on the dotted page.

## Headings keep a clean rhythm

Notebook matches a familiar editorial type scale, so `h2` and `h3` stay readable
without shouting.

### A third-level heading

Body text sits at a comfortable 18px with relaxed line spacing.

## Code blocks

Inline `code` reads as a small recessed chip, and fenced blocks become panels:

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello from Notebook")
}
```

## Blockquotes

> Good design beats clever prompts every time. The page should get out of the way
> and let the writing carry the weight.
